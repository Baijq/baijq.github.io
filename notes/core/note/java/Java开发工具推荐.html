# Java 技术栈开发工具推荐

作为一名程序员，假如你有一台全新的电脑，你会装那些软件呢？

# 一、后端开发电脑硬件配置

✔️处理器 至少i5以上吧
✔️内存 笔记本至少16G，台式机可以升级到32G(可以多跑几个虚拟机，在跑几个微服务)
✔️硬盘 固态512G + 机械硬盘1T(各种资料)
✔️显卡 没啥要求
✔️屏幕 其实能用就行，2K啥的看钱了

买的时候最好选大厂的，去官网或者京东官方自营店等渠道
