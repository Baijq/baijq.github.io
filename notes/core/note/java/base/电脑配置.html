# 电脑配件

# 1.电脑主要部件

项目 描述
CPU 大脑，负责运算和数据处理
主板 工作平台，联通并安装着各个硬件
内存 草稿纸，临时存放当前正在运行和计算的数据，主板右上角DIMM插槽(大多数4个)
显卡 图形处理中心，渲染并输出画面给显示器，主板上PCIe插槽，外接供电(3组8pin接口)
硬盘：固态/机械 存放文件，系统和游戏等数据。机械，主板右下角SATA接口；固态，M.2接口
散热 风冷，水冷
电源 交流转直流，提供给各部件电源。分为非模组，半模组，全模组；24pin主板供电、4+4pinCPU供电、6+2pinPCIe供电(显卡)、SATA供电、大4pin(D)供电(灯条)
机箱 USB、开机按钮、耳机等接口

# 2.安装步骤

  • CPU安装：主要正反，三角标志等
  • 内存条，DIMM插槽，大多数主板都是4各插槽；一个内存条2插槽，两个插入2，4插槽，不推荐三条，四条插满即可
  • 固态硬盘，M.2接口，注意有金属散热片的撕掉塑料保护膜，距离CPU最近的优先插入，其他都是挂在南桥PCH上的
  • 风冷散热，参照说明书；导热硅脂不要太多；撕掉塑料保护膜。风扇接口:CPU_FAN
  • 机箱，注意IO挡板，显卡挡板，机箱螺丝柱
  • SATA硬盘，3.5寸 => HDD位置，2.5寸机械，2.5寸固态三种 => SSD位置，插入硬盘抽屉
  • 机箱风扇。SYS_FAN
  • 电源，接线
  • 显卡
  • 显示器 插独立显卡位插槽

接线:

供电线 主板供电 => 20+4=24pin； CPU供电 => 4+4=8pin； 数据线 USB3.0、AUDIO音频接口、USB2.0=>JUSB 开机2pin、重启2pin、开机指示灯1+1=2pin(正负极)、硬盘指示灯2pin(正负极)
