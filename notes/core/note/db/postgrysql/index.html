# PostgreSQL

PostgreSQL是一种关系型数据库管理系统 （RDBMS）用它的原话来说就是 The World's Most Advanced Open Source Relational Database

  1. 文档：https://www.postgresql.org/docs/ (opens new window)

  2. 基础操作

    -- 1.创建/删除数据库
createdb mydb
dropdb mydb

-- 2.访问数据库
psql mydb

SELECT version();
SELECT current_date;

-- 3.创建删除表
CREATE TABLE weather (
    city            varchar(80),
    temp_lo         int,           -- 最低温度
    temp_hi         int,           -- 最高温度
    prcp            real,          -- 湿度
    date            date
);
DROP TABLE tablename;

-- 4.CURD
INSERT INTO weather VALUES ('San Francisco', 46, 50, 0.25, '1994-11-27');
COPY weather FROM '/home/user/weather.txt'; -- 使用COPY从文本文件中装载大量数据。这种方式通常更快，因为COPY命令就是为这类应用优化的， 只是比INSERT少一些灵活性
UPDATE weather
SET temp_hi = temp_hi - 2,  temp_lo = temp_lo - 2
WHERE date > '1994-11-28';

    CREATE VIEW myview AS
    SELECT city, temp_lo, temp_hi, prcp, date, location
	FROM weather, cities
	WHERE city = name;

SELECT * FROM myview;

  5. 事务

    在PostgreSQL中，开启一个事务需要将SQL命令用BEGIN和COMMIT命令包围起来，一组被BEGIN和COMMIT包围的语句也被称为一个事务块。

    BEGIN;
UPDATE accounts SET balance = balance - 100.00
    WHERE name = 'Alice';
-- etc etc
COMMIT;

SAVEPOINT my_savepoint;
ROLLBACK TO my_savepoint;
