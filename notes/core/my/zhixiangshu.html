# 1️⃣《致橡树》· 舒婷

# 我如果爱你————

# 绝不像攀援的凌霄花，

# 借你的高枝炫耀自己；

不会因为他的家庭地位去高攀

# 我如果爱你————

# 绝不学痴情的鸟儿，

# 为绿荫重复单调的歌曲；

不能一味地沉浸在情情爱爱中，丢弃自我。不能单方面的一味付出，这不是爱情

# 也不止像泉源，

# 常年送来清凉的慰藉；

每天关心呵护，一直在身边，需要的时候一直在

# 也不止像险峰，

# 增加你的高度，衬托你的威仪；

彼此当做榜样，一起成长，变优秀

# 甚至日光，

# 甚至春雨。

风风雨雨，也会有不开心，但是更多的是开心。必经风雨之后，才会有彩虹。

# 不，这些还不够！

# 我必须是你近旁的一株木棉，

# 作为树的形象和你站在一起。

人格平等，一直是坚韧而又温柔的合作伙伴

# 根，紧握在地下，

# 叶，相融在云里。

根是基础，彼此心心念念。叶与也的扶持变的更多更好；同享一片天，共同孕育在脚下的土地里

# 每一阵风过，

# 我们都互相致意，

# 但没有人

# 听懂我们的言语。

喜忧难，回头看， 致敬一路走过的路。没有人理解我们的经历，是唯一

# 你有你的铜枝铁干，

# 像刀，像剑，也像戟；

你有你的生活空间，一直向前支撑者一家，很锋利。

# 我有我红硕的花朵，

# 像沉重的叹息，

# 又像英勇的火炬。

我也有我的生活空间，微笑却又是生活中不可或缺，看不到的付出

# 我们分担寒潮、风雷、霹雳；

我们一起承担很大的压力，有大有小的事情

# 我们共享雾霭、流岚、虹霓。

一起享受奋斗来的成功

# 仿佛永远分离，

# 却又终身相依。

成年人的世界是孤独的，但是转身还有个你

# 这才是伟大的爱情，

# 坚贞就在这里；

# 爱————

# 不仅爱你伟岸的身躯，

# 也爱你坚持的位置，

# 足下的土地。
