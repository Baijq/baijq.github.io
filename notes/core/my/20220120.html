# 2️⃣ 触动心灵的几句话

愿所有离家漂泊在外的人，一路平安！找到知己！！

# 有家的地方没有工作，

# 有工作的地方没有家。

# 他乡容不下灵魂，

# 故乡安置不了肉身。

# 从此，便有了漂泊，有了乡愁，有了牵挂……

农村人，打工人的无奈...人人都想实现自身价值，都想到 大城市发展，可是过程中的辛酸只有自己知道。家乡没有合适的工作只能选择外出发展，俗话说“父母在不远游”，在自己的家乡才能感受到亲情的温暖，孤身在外总是孤单寂寞。特别是到了节假日，人家合家团圆你却孤家寡人，很失落的。

推荐的书

《悬浮》· 严飞

2022年广西师范大学出版社出版的图书

异乡人的都市生活
